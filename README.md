# yabc

> Yet Another Bluesky CLI

## Installation

### Using Go

```bash
go install github.com/alexisbcz/yabc@latest
```

### Using Goblin.run

```bash
curl -sf http://goblin.run/github.com/alexisbcz/yabc | sh
```

## Documentation

For complete documentation, run:

```bash
yabc help
```
